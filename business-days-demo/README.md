# Business Days Demo

Small Python utility for calculating a date after a number of business days.

This project is intentionally tiny so an agent can demonstrate a bug-fix loop:
read a GitHub issue, reproduce the bug with a test, patch the code, and verify
the fix.

## Run

```bash
python business_days.py 2026-06-05 1
```

## Test

```bash
python -m pytest tests
```
